# My Webpack 4 babel 7 setups

This setup contains Webpack 4,  Babel 7, Sass configurations with React-Redux.

## Installing

* Clone this project (git clone ...)
* Install npm dependencies (npm install)

## Build
npm run build

## Run dev-mode
npm run start
